# simple-twitter-project
以前後端分離方式進行的協作專案(前端部分)。
主要以vue進行開發，並使用vuex來優化登入狀態。
後端API部分：https://github.com/LIN-CHANG-YI/twitter-api-2020-backend

## 網站功能
- 一般使用者可新增推文、回覆推文。
- 一般使用者可對喜歡/取消喜歡推文。
- 一般使用者可追蹤/取消追蹤其他使用者。
- 一般使用者可更改自己的個人資料及帳號。
- 一般使用者可瀏覽單篇推文。
- 一般使用者可以查看其他使用者的個人頁面。
- 管理者可瀏覽所有一般使用者的名單。
- 管理者可瀏覽所有一般使用者的推文。
- 管理者可刪除一般使用者的任何一篇推文。

## 安裝/啟動方式

```
npm install
```
```
npm run serve
```
